# 📄 PDF Simples

Um conversor de PDF simples, prático e intuitivo, especialmente projetado para idosos. Sem complicações, sem login, direto ao ponto!

## ✨ Características

- **Interface Simples**: Botões grandes, texto legível, navegação intuitiva
- **Sem Login**: Use imediatamente, sem cadastros ou complicações
- **Conversão Rápida**: Converte imagens, textos e documentos para PDF
- **Fácil Compartilhamento**: Salve ou compartilhe seus PDFs diretamente
- **Design Acessível**: Pensado especialmente para idosos

## 🚀 Funcionalidades

- ✅ Converter imagens (JPG, PNG, etc.) para PDF
- ✅ Converter arquivos de texto para PDF
- ✅ Converter documentos Word para PDF
- ✅ Interface com botões grandes e texto legível
- ✅ Instruções claras passo a passo
- ✅ Compartilhamento direto do PDF gerado

## 📱 Como usar

1. **Selecionar Arquivo**: Toque no botão azul "SELECIONAR ARQUIVO"
2. **Escolher**: Selecione uma imagem, texto ou documento
3. **Converter**: Toque no botão verde "CONVERTER PARA PDF"
4. **Salvar**: Escolha onde salvar ou compartilhar seu PDF

## 🛠️ Instalação e Execução

### Pré-requisitos
- Node.js (versão 18 ou superior)
- Expo CLI
- Android Studio (para Android) ou Xcode (para iOS)

### Passos para executar

1. **Instalar dependências**:
   ```bash
   npm install
   ```

2. **Iniciar o projeto**:
   ```bash
   npm start
   ```

3. **Executar no dispositivo**:
   - **Android**: `npm run android`
   - **iOS**: `npm run ios`
   - **Web**: `npm run web`

### Para produção

1. **Instalar EAS CLI**:
   ```bash
   npm install -g @expo/eas-cli
   ```

2. **Fazer login no Expo**:
   ```bash
   eas login
   ```

3. **Configurar build**:
   ```bash
   eas build:configure
   ```

4. **Gerar APK para Android**:
   ```bash
   eas build --platform android --profile preview
   ```

## 📦 Dependências Principais
